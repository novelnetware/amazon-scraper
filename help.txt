## نحوه استفاده از معماری جدید
برای جمع‌آوری و ذخیره دسته‌بندی‌ها (فقط یک بار اجرا کنید):

Bash

go run ./cmd/scraper -task=scrape-categories
حالا باید لاگ‌های دقیق از پیدا شدن و ذخیره شدن هر دسته‌بندی را ببینید.

برای اسکرپ کردن محصولات:

Bash

go run ./cmd/scraper -task=scrape-products
برنامه لیست دسته‌بندی‌ها را به شما نشان داده و منتظر انتخاب شما می‌ماند.

برای اجرای سرور API:

Bash

go run ./cmd/scraper -task=server
این ساختار جدید به شما کنترل کامل بر روی فرآیندها می‌دهد و کد را بسیار تمیزتر و قابل مدیریت‌تر می‌کند.
